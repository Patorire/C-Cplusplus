C/C++

C es un lenguaje de programación (uno de los más importentes en la actualidad) con el cual se pueden desarrollar tanto aplicaciones como sistemas operativos a la vez que forma la base de otros lenguajes más actuales como Java, C++ o C#.

¿Cuál es su origen?

Pocos años después de la creación del sistema operativo UNIX (1969) en 1972 sucedió el nacimiento de este lenguaje de programación, de la mano del científico informático Dennis Ritche (junto con Ken Thompson). El nombre de C es simplemente por una cuestion alfabética, ya que sustituyó a sus lenguajes predecesores B y BCPL que codificaban la minicomputadora PDP-7.
C toma protagonismo para la creación del sistema operativo UNIX que traeria los nuevos minicomputadores PDP-11, los primeros en conectar el procesador, los periféricos y la memória entre sí mediante un BUS bidireccional.

Características de C

  • Estructura de C-Lenguaje estructurado.
  • Programación de nivel medio (beneficiándose de las ventajas de la programación de alto y bajo nivel).
  • No depende del hardware, por lo que se puede migrar a otros sistemas.
  • No es un lenguaje para una tarea específica, se puede programar tanto un sistema operativo como una hoja de cálculo o un juego.
  • Control absoluto de lo que sucede en el ordenador.
  • Organización del trabajo con toda libertad.
  • Rico en tipo de datos, operadores y variables en C

Sintaxis en C

La sintaxis en C son una serie de reglas que lideran la estructura del programa. Estas reglas son entendidas por el compilador, permitiendo crear un programa en C válido.
La sintaxis básica en C determina la forma en que se agrupan los carácteres para formar tokens que son la unidad mínima de programación en C. Estos tokens son los pequeños bloques con los que se programa en C, pudiendo ser variables, identificadores, constantes, palabras clave, simbolos que comprenda el lenguaje o sentencias en lenguaje C.
Para comenzar con el programa primero deberemos añadir bibliotecas a este con el comando #include más el archivo a vincular, por ejemplo si queremos utilizar dentro del programa la orden printf primero deberemos vincular al #include el archivo stdio.h.
La función int main() hace que el programa vuelva a ese punto tras ejecutarse y su orden esta limitada entre corchetes {}.

Bibliotecas de C

Cuando hablamos de librerías que son usadas en este lenguaje de programación y están formadas por una serie de código con el que podemos leer el teclado, realizar operaciones matemáticas, imprimir en pantalla... La bibilioteca referente es la estándar de ANSI C formada por 24 archivos de cabecera. Segun su función se clasifican en categorías, como por ejemplo matemáticas, de salida y entrada, de uso de textos, de gestión de memoria...
Las librerías más destacadas son stdio.h que es de entrada y salida, conio.h para la gestión de textos, math.h encargada de operaciones matemáticas, time.h obtiene fecha y hora actual para poder utilizarla según sus necesidades, etc...

¿Qué es C++

C++ es un lenguajed de programación orientado a objetos muy potente que ha evolucionado como extensión del lenguaje C.
Es una de las herramientas programáticas más versátiles y completas, por esa razón un programador C++ con experiencia es un perfil profesional con mucha demanda en todo tipo de proyectos tecnológicos.

  • Bases de datos

  • Apps móviles

¿Cómo funciona C++

C++ fue desarrollado por Bjarne Stroustrup en 1979. Este lenguaje se caracteriza por usar un código que no solo se basa en números y letras, también en reglas lógicas de sintaxis y variables similares a las del lenguaje humano.

Ventajas y desventajas de C++

Ventajas

  • Se pueden reutilizar bloques completos de código para desarrollar programas nievos en menos tiempo.
  • Puede usarse para el desarrollo de diversas aplicaciones, además es multiplataforma.
  • Genera un código compacto y fácil de correr.

Desventajas

  • Interfaza poco intuitiva.
  • Facilidad al cometer errores.
  • Su curva de aprendizaje es de las más largas.
  • No es útil para crear sitios web.
